== Description ==
DupleXer is a simple plugin that allows you to duplicate posts, pages, and WooCommerce products with one click.

== Installation ==
1. Upload the `duplexer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Does it support WooCommerce? =
Yes, you can duplicate WooCommerce products with this plugin.
